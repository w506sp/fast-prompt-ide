fast-prompt-ide

a minimalist ide for managing, testing, and versioning prompts against local llms via ollama.

log in or sign up to get started.

features

  • workspaces & projects — organize prompts and share them with teammates.
  • versioning — every prompt change creates a numbered version with a commit message.
  • execution — run a version against any installed ollama model, with {{variable}} substitution.
  • history & metrics — every run records latency and token usage.
